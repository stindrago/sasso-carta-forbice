# Sasso Carta Forbice

Il classico gioco Sasso Carta e Forbice creato in LISP per l'[Hacktoberfest 2019][hacktoberfest-2019]

Per poter giocare:
1. Installare SBCL:
    + Per Arch: `sudo pacman -S sbcl`
    + Per Debian: `apt-get install sbcl`
1. Eseguire il file [`scf.lisp`][scf-lisp]:
    + `sbcl --load scf.lisp`
